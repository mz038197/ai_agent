# 歡迎使用 AI 助手 👋

這是一個基於 Ollama 和 Chainlit 構建的智能對話助手。

## 功能特色 ✨

- 💬 **智能對話**: 支援自然語言對話
- 🖼️ **圖片理解**: 上傳圖片並提問
- 🚀 **本地運行**: 完全在本地運行，保護隱私
- ⚡ **即時回應**: 快速的回應速度

## 如何使用 📖

1. **文字對話**: 直接在輸入框輸入您的問題
2. **圖片分析**: 點擊輸入框旁的 📎 按鈕上傳圖片，然後輸入您的問題

## 技術棧 🛠️

- **模型**: Gemma 3 (4B 參數)
- **框架**: LangChain + Chainlit
- **後端**: Ollama

開始對話吧！👇
